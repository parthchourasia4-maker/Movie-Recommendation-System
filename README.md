# 🎬 Movie Recommendation System

A Content-Based Movie Recommendation System built using Python and Machine Learning techniques. The project recommends movies similar to a user's favorite movie based on textual features.

## 📌 Features

- Content-Based Recommendation
- TF-IDF Vectorization
- Cosine Similarity
- Data Cleaning & Preprocessing
- Interactive recommendation function

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset

This project uses movie metadata including titles, genres, cast, crew and overview to generate recommendations.

## ⚙️ How It Works

1. Load and preprocess movie data.
2. Clean textual information.
3. Convert text into numerical vectors using TF-IDF.
4. Calculate cosine similarity between movies.
5. Recommend the most similar movies based on the selected title.
